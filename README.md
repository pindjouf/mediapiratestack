# Media Pirate Stack
**Hardware needed:**
- SBC = [Raspberry Pi](https://www.amazon.com.be/-/en/dp/B09TTNF8BT?ref=ppx_yo2ov_dt_b_product_details), [ZimaBlade](https://shop.zimaboard.com/products/zimablade-single-board-server-for-cyber-native)
- Storage = [SSD](https://www.amazon.com.be/-/en/Kingston-NVMe-Internal-SNV2S-1000G/dp/B0BBWH1R8H/ref=sr_1_8?sr=8-8), [External SSD](https://www.amazon.com.be/s?k=ssd+usb&ref=nb_sb_noss_1)

**Services needed:**
- Torrenting client = [Transmission](https://transmissionbt.com)
- Media Finder Interface = [Radarr](https://radarr.video/), [Sonarr](https://sonarr.tv/), [Lidarr](https://lidarr.audio/)
- API support for torrent trackers (optional) = [Jackett](https://github.com/Jackett/Jackett)
- VPN = [WireGuard](https://www.wireguard.com/), [openvpn](https://openvpn.net/), [pivpn](https://www.pivpn.io/)
- Media Player/Center = [VLC](https://www.videolan.org/), [Jellyfin](https://jellyfin.org/), [Kodi](https://kodi.tv/)
